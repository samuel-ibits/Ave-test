# Book API

This is a RESTful API built using Node.js and Express.js for managing a list of books. The API supports CRUD (Create, Read, Update, Delete) operations.

## Installation

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd book-api
   ```

### To run

Type in terminal
`npm install`
`node app.js`
